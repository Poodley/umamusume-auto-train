# Umamusume Auto Train

This is a fork of an early iteration of https://github.com/samsulpanjul/umamusume-auto-train. Many thanks to those who made that repository for creating the base code. This fork offers several improvements to runtime.

## Features

- Training hints for spark farming: simply screenshot the support card hint icon into assets/hint_icons, and add the hints you want into logic.py and path to the icon in state.py
